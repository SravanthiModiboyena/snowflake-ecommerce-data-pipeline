# Snowflake E-commerce Data Pipeline

## Project Overview
This project implements a Medallion Architecture (Bronze, Silver, Gold)
using Snowflake.

## Technologies Used
- Snowflake
- SQL
- Git
- GitHub

## Architecture
CSV → RAW → SILVER → GOLD
Using Streams and Tasks for automation.

## Features
- Data cleaning
- Business aggregation
- Automated pipeline
